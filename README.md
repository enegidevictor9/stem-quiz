# STEM Questions Quiz - 60 Questions Edition

An interactive quiz application featuring 60 comprehensive STEM (Science, Technology, Engineering, Mathematics) questions with multiple-choice answers.

## Features

- ✨ **Interactive Quiz Interface** - Click any question button to answer
- 📊 **Score Tracking** - Monitor your progress with the score counter
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Modern UI** - Beautiful gradient design with smooth animations
- ✅ **Instant Feedback** - Get immediate feedback on your answers
- 🔄 **Retakeable** - Answer questions as many times as you want
- 📝 **Visual Progress** - Answered questions are marked with a checkmark

## Question Categories

### Chemistry (10 Questions)
- Water formula, atomic numbers, states of matter, pH, elements, and more

### Physics (10 Questions)
- Speed of light, Newton's laws, energy, waves, sound, and more

### Biology (15 Questions)
- Photosynthesis, cell biology, human anatomy, evolution, genetics, and more

### Mathematics (10 Questions)
- Geometry, algebra, calculus, trigonometry, and more

### Earth Science (10 Questions)
- Solar system, atmosphere, geology, weather, and climate

### Technology & Engineering (5 Questions)
- Computing, electricity, chemistry, and engineering principles

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. Click on any question button to open the quiz modal
4. Select an answer from the options provided
5. Click "Submit Answer" to check if your answer is correct
6. View the feedback and track your score
7. Use "Reset All" button to start over

## Files

- `index.html` - Main HTML structure with question grid
- `style.css` - Styling, animations, and responsive layout
- `script.js` - JavaScript functionality with all 60 questions
